# Templates

## English

Reusable, copy-paste working documents for applying the playbook in a
repository. They are tool-agnostic: adapt them to your harness, CI, and team.

| Template | Purpose |
|---|---|
| [`spec-with-evals.template.md`](./spec-with-evals.template.md) | A spec that also states how correct behavior is recognized (evals). |
| [`task-slicing.template.md`](./task-slicing.template.md) | Split a larger spec into small, independently reviewable tasks. |
| [`agent-task.template.md`](./agent-task.template.md) | One focused task for a single implementation agent. |
| [`review-report.template.md`](./review-report.template.md) | Review a change against spec, evals, and risk, not just "looks good". |
| [`pr-description.template.md`](./pr-description.template.md) | Make AI involvement, risk tier, evals, and review level visible in the PR. |
| [`risk-classification.template.md`](./risk-classification.template.md) | Classify a task's risk before an agent starts. |

Each template has a German companion with the `.de.md` suffix.

`pr-description.template.md` is a standalone artifact to copy into your own
repo. It does not replace this repository's `.github/pull_request_template.md`.

## Deutsch

Wiederverwendbare Arbeitsdokumente zum Kopieren, um das Playbook in einem
Repository anzuwenden. Sie sind tool-agnostisch: passe sie an deinen Harness,
deine CI und dein Team an.

| Vorlage | Zweck |
|---|---|
| [`spec-with-evals.template.de.md`](./spec-with-evals.template.de.md) | Eine Spec, die auch festhält, woran korrektes Verhalten erkannt wird (Evals). |
| [`task-slicing.template.de.md`](./task-slicing.template.de.md) | Eine größere Spec in kleine, unabhängig reviewbare Tasks schneiden. |
| [`agent-task.template.de.md`](./agent-task.template.de.md) | Ein fokussierter Auftrag für einen einzelnen Implementierungs-Agenten. |
| [`review-report.template.de.md`](./review-report.template.de.md) | Eine Änderung gegen Spec, Evals und Risiko prüfen, nicht nur „sieht gut aus“. |
| [`pr-description.template.de.md`](./pr-description.template.de.md) | KI-Beteiligung, Risk Tier, Evals und Review-Level im PR sichtbar machen. |
| [`risk-classification.template.de.md`](./risk-classification.template.de.md) | Das Risiko eines Tasks klassifizieren, bevor ein Agent startet. |

Jede Vorlage hat ein englisches Gegenstück ohne das `.de.md`-Suffix.

`pr-description.template.de.md` ist ein eigenständiges Artefakt zum Kopieren in
dein eigenes Repo. Es ersetzt nicht das `.github/pull_request_template.md`
dieses Repositories.
