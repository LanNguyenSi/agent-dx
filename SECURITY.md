# Security Policy

## Supported Versions

Active development is on `master`. Only the latest release of each `packages/<tool>` is supported.

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security reports.

Email **raylsreturn@googlemail.com** with:

- Affected package and version
- Reproduction steps or proof-of-concept
- Impact assessment

You will get an acknowledgement within 72 hours and a fix timeline within 7 days.
